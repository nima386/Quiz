STYLE:     Dark Glassmorphism · Indigo-Purple · Minimal · Smooth
VIBE:      Modern Quiz-App · Lernplattform · Motivierend · Premium
REFERENCE: Midnight-blue UI mit purple glow, ähnlich wie dark finance dashboards

// ── GLOBALE REGELN ────────────────────────────
RULE 1: Niemals weiße Hintergründe. Immer dark.
RULE 2: Jede interaktive Fläche hat einen hover-State.
RULE 3: Zahlen immer mit JetBrains Mono.
RULE 4: Radius immer >= 10px. Keine harten Ecken.
RULE 5: Gradient-Buttons haben immer einen matching glow-shadow.
RULE 6: Icons: Outline-Stil, Stroke-Width 1.5–2px, Farbe text-muted oder primary-light.
RULE 7: Kein reines Schwarz (#000) und kein reines Weiß (#fff) im UI.
RULE 8: Abstände in 4px-Raster (4, 8, 12, 16, 20, 24, 32, 40, 48).

03 — so nutzt du diesen prompt
/* ─── IN CHATGPT / CLAUDE / CURSOR / V0 / BOLT ─── */

BEISPIEL-PROMPT für eine neue Screen:

"Erstelle einen Quiz-Screen für meine App.
Verwende folgendes Design-System:

background: #0f0f1a → #1c1c35 (cards)
primary: #6e63f2, accent-pink: #f06292, text: #f0eeff/#9190b8
font: Outfit (headings), JetBrains Mono (zahlen/stats)
border-radius: 22px (cards), 9999px (buttons/badges)
buttons: gradient primary mit glow-shadow
cards: dark mit subtle purple-gradient innen
stats: mono font, grün für +, rot für −
badges: pill-style mit semi-transparent hintergrund
progress: 6px track, gradient fill

Der Screen soll zeigen: [DEINE BESCHREIBUNG]"

/* ─── FÜR CSS CUSTOM PROPERTIES (copy-paste ready) ─── */

:root {
  --bg-base: #0f0f1a; --bg-surface: #161628;
  --bg-card: #1c1c35; --bg-elevated: #252545;
  --primary: #6e63f2; --primary-light: #a29bfe;
  --pink: #f06292; --teal: #26c6da;
  --amber: #ffca28; --green: #26d47c; --red: #f06060;
  --text-1: #f0eeff; --text-2: #9190b8; --text-3: #4e4d72;
  --border: rgba(255,255,255,0.10);
  --border-focus: rgba(110,99,242,0.60);
  --glow: 0 0 24px rgba(110,99,242,0.35);
  --r-card: 22px; --r-btn: 9999px; --r-input: 16px;
}
